# BitoGuard Demo 快速開始 - 真實 API 版

## 🚀 快速啟動（3 步驟）

### 1. 安裝依賴

```bash
pip install -r requirements.txt
```

### 2. 啟動 Demo

**Windows:**
```bash
run_real_api_demo.bat
```

**Linux/Mac:**
```bash
chmod +x run_real_api_demo.sh
./run_real_api_demo.sh
```

**或直接使用 Streamlit:**
```bash
streamlit run app.py
```

### 3. 開始使用

1. 瀏覽器會自動開啟 `http://localhost:8501`
2. 在搜尋框輸入 User ID（例如：`12345`）
3. 點擊「🔍 查詢」按鈕
4. 查看風險分析結果！

## 📊 功能說明

### 查詢真實用戶
- 輸入任何 BitoPro User ID
- 系統會從真實 API 獲取資料
- 自動計算風險分數
- 顯示詳細特徵分析

### 查看總覽
- 點擊「📊 總覽」按鈕
- 查看模擬資料的統計圖表
- 瀏覽可疑帳號列表
- 分析風險分布

## 🔧 疑難排解

### 問題：API 連線失敗
**解決方案：**
```bash
# 測試網路連線
python test_real_api.py

# 檢查防火牆設定
# 確保可以存取 https://aws-event-api.bitopro.com
```

### 問題：模型訓練太慢
**解決方案：**
- 等待首次訓練完成（只需一次）
- 或修改程式碼減少訓練樣本數

### 問題：找不到用戶
**解決方案：**
- 確認輸入的是純數字 User ID
- 嘗試其他 User ID
- 檢查 API 資料是否正常載入

## 📝 注意事項

1. ✅ 需要網路連線（存取 BitoPro API）
2. ✅ 首次啟動會較慢（載入資料 + 訓練模型）
3. ✅ 資料會快取 1 小時
4. ✅ 只使用部分訓練資料以加快速度
