## 👋 歡迎！

---

## ⚡ 3 步驟快速啟動

### 1️⃣ 安裝依賴
```bash
pip install -r requirements.txt
```

### 2️⃣ 測試環境
```bash
python test_demo.py
```

應該看到：
```
✅ All tests passed! Demo is ready to run.
```

### 3️⃣ 啟動 Demo
```bash
streamlit run app.py
```

Demo 將在瀏覽器自動開啟：`http://localhost:8501`
