## 🎯 使用場景導航

### 場景 1: 第一次使用
1. 閱讀 [QUICKSTART.md](QUICKSTART.md)
2. 執行 `python test_demo.py`
3. 執行 `streamlit run app.py`
4. 瀏覽 Demo 介面

### 場景 2: 準備比賽展示
1. 閱讀 [DEMO_GUIDE.md](DEMO_GUIDE.md)

3. 準備計時器和備用方案
4. 檢查展示檢查清單

1. 閱讀 [README.md](README.md) 了解功能
2. 閱讀 [DEMO_GUIDE.md](DEMO_GUIDE.md) 準備腳本
3. 根據客戶需求調整展示重點
4. 準備回答常見問題

### 場景 4: 開發擴展
1. 閱讀 [README.md](README.md) 的「後續整合」章節

### 場景 5: UI 設計參考

## 🎉 快速命令

### 測試環境
```bash
python test_demo.py
```

### 啟動 Demo
```bash
streamlit run app.py
```

### 安裝依賴
```bash
pip install -r requirements.txt
```

### Windows 一鍵啟動
```bash
run_demo.bat
```

### Linux/Mac 一鍵啟動
```bash
chmod +x run_demo.sh
./run_demo.sh
```
